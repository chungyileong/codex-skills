# Codex Skills Collection

## Skills

- [`executor-advisor`](skills/executor-advisor/) — coordinates a typically
  low-cost root executor with a persistent, read-only advisor for complex or
  review-heavy work.

## Install

Ask Codex to use the official skill installer, replacing `<skill-name>` with a
skill listed above:

```text
Use $skill-installer to install
https://github.com/chungyileong/codex-skills/tree/main/skills/<skill-name>
```
